Asynchronous HTTP client¶

Pulsar has a HttpClient for asynchronous HTTP requests:

>>> from pulsar.apps import http
>>> client = http.HttpClient()
>>> response = http.get('http://www.bbc.co.uk')

Features¶

  • No dependencies
  • Similar API to requests
  • Cookie support
  • Web socket support
  • Thread safe
  • Can be used in synchronous mode

Making requests¶

To make a request is simple:

from pulsar.apps import http
client = http.HttpClient()
resp = client.get('https://github.com/timeline.json')

resp is a HttpResponse object which contains all the information about the request and, once finished, the result.

The resp is finished once the on_finished attribute (a pulsar.Deferred) is fired. In a coroutine one can obtained a full response by yielding on_finished:

resp = yield client.get('https://github.com/timeline.json').on_finished

Cookies¶

Cookies are handled by the client by storing cookies received with responses. To disable cookie one can pass store_cookies=False during HttpClient initialisation.

API¶

The main class is the HttpClient which is a subclass of pulsar.Client.
