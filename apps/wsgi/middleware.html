Middleware¶

This module implements several WSGI middleware which does not serve request but instead perform initialization and sanity checks. It also introduces the Response middlewares implemented as a subclass of ResponseMiddleware. Response middlewares are used by the pulsar.apps.wsgi.wrappers.WsgiResponse to modify/add headers and manipulate content.

Additional WSGI Middlewares¶

Several wsgi middleware useful in several applications.

clean path¶

pulsar.apps.wsgi.middleware.clean_path_middleware(environ, start_response)[source]¶

Clean url from double slashes and redirect if needed.

cookie¶

pulsar.apps.wsgi.middleware.cookies_middleware(environ, start_response)[source]¶

Parse the HTTP_COOKIE key in the environ and set the new http.cookie key in environ with a dictionary of cookies obtained via the pulsar.utils.httpurl.parse_cookie() function.

authorization¶

pulsar.apps.wsgi.middleware.authorization_middleware(environ, start_response)[source]¶

Parse the HTTP_AUTHORIZATION key in the environ if available and set the http.authorization key in environ with the result obtained from pulsar.apps.wsgi.plugins.parse_authorization_header() function.

Response Middlewares¶

Response middleware are callable objects which can be used in conjunction with pulsar application handlers. They must return a WsgiResponse which can be the same as the one passed to the callable or a brand new one.

Interface¶

class pulsar.apps.wsgi.middleware.ResponseMiddleware[source]¶

Bas class for pulsar.apps.wsgi.wrappers.WsgiResponse middlewares. The focus of this class is the execute() method where the middleware logic is implemented.

available(environ, response)[source]¶

Check if this ResponseMiddleware can be applied to the response object.

Parameters:
  • environ – a WSGI environ dictionary.
  • response – a pulsar.apps.wsgi.wrappers.WsgiResponse
Returns:

True or False.
execute(environ, response)[source]¶

Manipulate response, called only if the available() method returns True.

GZip Middleware¶

class pulsar.apps.wsgi.middleware.GZipMiddleware(min_length=200)[source]¶

A ResponseMiddleware for compressing content if the request allows gzip compression. It sets the Vary header accordingly.

The compression implementation is from http://jython.xhaus.com/http-compression-in-python-and-jython
