Gloabl Events¶

Multi-consumer multi-producer dispatching mechanism, originally based on pydispatch (BSD).

To register a new event use the bind() function:

from pulsar.utils import events

def handler(signal, sender, data=None, **params):
    ...

events.bind('connection_lost', handler)
