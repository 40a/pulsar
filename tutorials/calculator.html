JSON-RPC Calculator¶

The code for this example is located in the examples.calculator.manage module.

This is a a JSON-RPC server with some simple functions. To run the server type:

python manage.py

Open a new shell and launch python and type:

>>> from pulsar.apps import rpc
>>> p = rpc.JsonProxy('http://localhost:8060', force_sync=True)
>>> p.ping()
'pong'
>>> p.functions_list()
[[...
>>> p.calc.add(3,4)
7.0

The force_sync keyword is used here to force the Json RPC client to wait for a full response rather than returning a pulsar.Deferred. Check the creating synchronous clients tutorial.

Implementation¶

The calculator rpc functions are implemented by the Calculator handler, while the Root handler exposes utility methods from the pulsar.apps.rpc.PulsarServerCommands handler.
