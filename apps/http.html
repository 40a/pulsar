Asynchornous HTTP client¶

An HttpClient for asynchronous HTTP requests. It can also be used in synchronous mode by passing force_sync during initialization:

>>> from pulsar.apps import http
>>> client = http.HttpClient()
>>> response = http.get('http://www.bbc.co.uk')
