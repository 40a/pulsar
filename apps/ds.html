Pulsar Data Store Server¶

Pulsar-ds is a python implementation of the popular redis data store. It uses pulsar asynchronous framework to create a single-threaded workers responding to TCP-requests in the same way as redis does.

To run a stand alone server create a script with the following code:

from pulsar.apps.data import PulsarDS

if __name__ == '__main__':
    PulsarDS().start()

More information on the pulsar data store example.
