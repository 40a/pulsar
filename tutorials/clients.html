Writing Clients¶

The code for this example is located in the examples.echo.manage module.

Pulsar provides several classes for writing clients. The first step is to subclass pulsar.ProtocolConsumer which is needed for two reasons:

  • It encodes and sends the request to the remote server.
  • It listen (if needed) for incoming data from the remote server.

There are two methods which needs implementing:

  • pulsar.ProtocolConsumer.start_request() to kick start a request to a remote server.
  • pulsar.Protocol.data_received(), invoked by the pulsar.Connection when new data has been received from the remote server.

In this example we implement a very simple Echo server/client pair. Because this example has symmetric client and server protocols, the EchoProtocol will also be used for the server.

Echo Protocol¶

class examples.echo.manage.EchoProtocol(*args, **kwargs)[source]¶

A symmetric echo protocol consumer for client and servers. The only difference between client and server is the implementation of the response() method.

separator¶

A separator for messages.

data_received(data)[source]¶

Implements the pulsar.Protocol.data_received() method. It simply search for the separator and, if found, it invokes the response() method with the value of the message.

response(data)[source]¶

Clients return the message so that the pulsar.ProtocolConsumer.on_finished deferred is called back with the message value, while servers sends the message back to the client.

Echo client

alias of EchoProtocol

request(message)[source]¶

Build the client request and return a pulsar.Deferred which will be called once the server has sent back its response.

Run The example¶

To run the server:

python manage.py

Open a new shell, in this directory, launch python and type:

>>> from manage import Echo
>>> p = Echo('localhost:8060', force_sync=True)

The force_sync set to True, force the client to wait for results rather than returning a pulsar.Deferred. Check the creating synchronous clients tutorial for further information.

>>> p.request('Hello')
'Hello'
