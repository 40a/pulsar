Websocket Frame & Parser¶

WebSocket Protocol Frame and FrameParser classes. These two classes can be used for both clients and server protocols.

Frame¶

class pulsar.utils.websocket.Frame(message=None, opcode=None, version=None, masking_key=None, final=False, rsv1=0, rsv2=0, rsv3=0, build_frame=True)[source]¶

Implements the framing protocol as defined by specification supporting protocol version 13:

>>> f = Frame('hello world', final=1)
>>> data = f.build()
>>> f = Frame()
>>> f.parser.send(bytes[1])
>>> f.parser.send(bytes[2])
>>> f.parser.send(bytes[2:])

Frame Parser¶

class pulsar.utils.websocket.FrameParser(version=None, kind=0)[source]¶

Decoder and encoder for the websocket protocol.

version¶

Optional protocol version (Default 13).

kind[source]¶
  • 0 for parsing client’s frames and sending server frames (to be used in the server)
  • 1 for parsing server frames and sending client frames (to be used by the client)
  • 2 Assumes always unmasked data
encode(data, final=True, masking_key=None, **params)[source]¶

Encode data into a Frame.

Parameters:
  • final – Indicating if this is a final Frame.
  • masking_key – Optional making key used only if kind is 1 (Client frames).
ping(body=None)[source]¶

return a ping Frame.

pong(body=None)[source]¶

return a pong Frame.

close(body=None)[source]¶

return a close Frame.

continuation(body=None)[source]¶

return a continuation Frame.

decode(data=None)[source]¶

Decode bytes data into a Frame. If kind is 0 it descodes into a client frame (masked frame) while if is 1 it decodes into a server frame (unmasked).
